About the Project

BranchOff is a multiplayer gamification app designed to revolutionize how parents manage their children's screen time. Unlike traditional restriction-based approaches, BranchOff transforms screen time management into an engaging challenge system.

As Co-Founder and Full-Stack Developer, I architected and built the complete technical infrastructure‚Äîfrom Figma designs to production-ready Swift code, Node.js backend synchronization, and Firebase database management. The app leveraged iOS's native Screen Time API to create a seamless, real-time challenge experience between parents and children.

The core innovation was our challenge-based system: parents create custom challenges with specific durations, app restrictions, and rewards. Children can accept these challenges, and both parties receive live updates through push notifications. If a child completes the challenge, they earn the reward; if they forfeit by opening a blocked app, they lose the reward but regain access immediately‚Äîputting choice and motivation at the center of the experience.

Key Features

Challenge System

Parents create custom challenges with specific durations, app blocks, and rewards. Children can accept or forfeit at any time, maintaining agency over their choices.

Live Synchronization

Real-time countdown timers on both parent and child devices, synchronized through Node.js backend for instant updates across all devices.

Push Notifications

Children can send status updates to parents. Automatic notifications when challenges are accepted, forfeited, or completed successfully.

iOS Screen Time API

Native integration with iOS Screen Time API for seamless app blocking across all of the child's devices without workarounds.

Real-Time Database

Firebase backend storing challenge data, user profiles, and game states with instant synchronization between family members.

Challenge Tracking

Complete history of challenges with status tracking (active, won, lost, forfeited) and detailed analytics for parents.

Technical Architecture

Frontend Development

Swift & UIKit

Designed complete user experience in Figma before implementation. Built dual interfaces for parent and child users with distinct feature sets. Implemented real-time countdown displays, challenge management interfaces, and push notification handling. Integrated iOS Screen Time API for native app blocking functionality.

Backend Infrastructure

Node.js Server

Developed Node.js backend to handle real-time data synchronization between devices. Managed bi-directional communication for instant updates when challenges are created, accepted, forfeited, or completed. Implemented push notification delivery system for cross-device updates.

Database Management

Firebase

Architected database schema for user profiles, active challenges, and historical data. Stored challenge parameters (duration, start/end times, blocked apps, rewards) and real-time status updates (active, won, lost, forfeited). Implemented efficient queries for instant data retrieval and updates.
